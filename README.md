# 🐍 贪吃蛇游戏

一个使用HTML、JavaScript和Tailwind CSS开发的现代化贪吃蛇游戏。

## ✨ 功能特性

- 🎮 **完整的游戏逻辑** - 经典的贪吃蛇玩法
- 📊 **积分系统** - 实时显示当前分数和最高分记录
- 🏆 **等级系统** - 每100分升一级，速度逐渐加快
- 🎨 **现代化UI** - 使用Tailwind CSS设计的优美界面
- 💾 **本地存储** - 最高分记录保存在浏览器本地
- 📱 **响应式设计** - 适配不同屏幕尺寸
- 🎯 **游戏控制** - 开始游戏、重新开始、游戏结束弹窗
- 🔊 **音效系统** - 移动、得分、失败、开始游戏音效

## 🎯 游戏规则

1. 使用方向键（↑↓←→）控制蛇的移动
2. 吃到红色食物获得10分
3. 撞墙或撞到自己身体游戏结束
4. 每100分升一级，速度会逐渐加快
5. 最高分记录会保存在浏览器中
6. 支持音效反馈，可随时开启/关闭

## 🚀 如何运行

1. 下载所有文件到本地目录
2. 用浏览器打开 `index.html` 文件
3. 点击"开始游戏"按钮开始游戏
4. 使用方向键控制蛇的移动

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **JavaScript ES6+** - 游戏逻辑
- **Canvas API** - 游戏渲染
- **Tailwind CSS** - 样式设计
- **LocalStorage** - 数据持久化
- **Web Audio API** - 音效生成

## 🔧 自定义选项

您可以通过修改 `game.js` 文件中的以下参数来自定义游戏：

- `gridSize` - 网格大小
- `speed` - 初始游戏速度
- 分数计算规则
- 等级提升条件
- 颜色主题
- 音效频率和音量
